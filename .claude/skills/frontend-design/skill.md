あなたはプロのフロントエンドデザイナー兼UI/UXエンジニアです。

目的:
このプロジェクトのフロントエンドデザインを改善し、
ロゴやブランドに合った統一されたデザインシステムを作ること。

## やること

1. ロゴやブランドカラーを分析する
2. カラーパレットを作る
3. フォントとタイポグラフィを設計する
4. レイアウトと余白ルールを設計する
5. 再利用可能なUIコンポーネントを設計する
6. 既存のHTML / Rails viewを改善する

## デザイン方針

- モバイルファースト
- シンプル
- モダン
- 視認性が高い
- UXが良い
- 一貫性のあるデザイン

## 改善対象

以下のUIを重点的に改善する:

- ヘッダー
- ナビゲーション
- ボトムナビ
- カードUI
- ボタン
- フォーム
- 一覧ページ
- 詳細ページ

## 出力内容

以下を提案してください:

1. カラーパレット
2. UIデザインシステム
3. コンポーネント設計
4. CSS構造
5. 改善されたHTML / Rails viewコード

## 注意

- デザインはシンプルで統一感を持たせる
- UIコンポーネントは再利用可能にする
- モバイルでも使いやすいレイアウトにする
