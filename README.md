# Kebab Tokyo

美味しいケバブに出会うための、ケバブ特化型レビュー共有アプリ

---

## **サービス概要**

美味しいケバブを食べたいけれど、住んでいるエリアや都内にどんなケバブ屋さんがあるのかを知ることができ、Googleマップに表示されていないお店も見つけることができる。

**Kebab Tokyo**は、そんなユーザーが実際に食べたケバブ屋さんを投稿・レビューを閲覧したり、写真・評価・場所とともにみんなと共有できる**ケバブ特化型レビューアプリ**です。

ケバブ好きな人からケバブを食べたことがない人にまで、ケバブ店に特化した飲食店の出会いを提供します。

---

## **このサービスへの思い・作りたい理由**

私自身、都内在住で外出先や帰り道でケバブ屋さんを見かけることが多く、

「このお店は前に食べたお店に比べて美味しいだろうか？」「マップには表示されていないがここにはケバブ屋さんが存在している！」と色々考えてワクワクしながら購入することが多々ありました。

SNS や Googleマップで調べてみても、

- ケバブ屋さんに関する店舗情報やレビューが少ない
- お店情報や写真が古い
- そもそもお店情報がネット上になく、レビューが見つからない

といったケースが多く、

**“ケバブ屋さんだけに特化した情報” が圧倒的に足りない** と感じていました。

また、ケバブは

- 店舗ごとに味や形・量の違いがある
- ソース・肉の種類・ボリューム感など店舗によって個性が強い

自身の経験上でこのような特徴があると感じており、それを気軽に共有できる場がほとんどありませんでした。

「それなら、世の中のケバブ好きが自然と集まり、美味しい店・面白い店・ちょっと微妙だった店も含めて正直に共有できる場所を作ろう」

そう思ったことがきっかけで、このアプリの開発を企画しました。

---

## **ユーザー層について**

| ユーザー層 | 理由 |
| --- | --- |
| ケバブが好きな男女(年代問わず) | 都市部を中心にケバブを食べる機会が多く、情報共有ニーズが高いため |
| 夜ごはん・軽食を外で済ませがちな社会人 | 失敗しない店選びの指標として役立つため |
| ケバブを食べたことはないが、興味がある人 | 食べようと思う行動のきっかけとして、豊富な店舗情報やレビューが参考になるため |

---

## **サービスの利用イメージ**

1. ユーザーがケバブ屋さんを見つけて来店
2. 食後にアプリから「お店に関するレビューを投稿」
    - 店名
    - 写真
    - 評価(点数評価)
    - コメント
    - 位置情報
3. 他のユーザーは地図や一覧からお店を探す
4. 「いいね」やコメントで交流が生まれる

→ **得られる価値**：

- 既存のマップ・ネット情報にはないケバブ店などの情報を得ることができる
- 初めての店でも失敗しにくくなる
- 自分が来店したケバブ店を記録・共有できる
- ケバブ好き同士のゆるいコミュニティが生まれる

## **ユーザーの獲得について**

- X（Twitter）での投稿・シェア機能
- Instagram での写真投稿との相性
- 「#ケバブ」「#ケバブ東京」などのハッシュタグ活用
- 個人ブログ・ポートフォリオでの宣伝

---

## **サービスの差別化ポイント・推しポイント**

| 差別化ポイント | 内容 |
| --- | --- |
| **① ケバブ特化型レビュー** | ケバブだけに集中した情報設計 |
| **② 豊富な店舗情報** | 既存のマップアプリにはない情報の入手 |
| **③ 地図ベースで探せるUX** | 今いる場所・近くのケバブ屋がすぐ分かる |
| **④ コミュニティ** | コメント・いいね機能付きで気軽に情報の共有・交流が可能 |

他アプリとの違い・推しポイント：

Google マップや一般的なグルメサイトは、

- 情報が多すぎる
- ケバブに特化していない
- お店情報として登録されていないケバブ店舗がある

という課題があります。

---

**Kebab Tokyo** は、

- ケバブの情報に特化している
- ケバブ好き・ケバブに興味を持った人がゆるっとしたコミュニティを形成できる機能設計
- 既存のマップやインターネット上にない情報が集まる

という設計になっています。
